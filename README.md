##### 其他
* :construction: - 正在进行中
* :ambulance: - 重要补丁
* :sparkles: - 引入新特性
* :memo: - 文档撰写
* :lipstick: - 更新UI和样式文件
* :lock: - 修复安全问题
* :rotating_light: - 修复Linter警告
* :chart_with_upwards_trend: - 添加分析或跟踪代码
* :globe_with_meridians: - 国际化和本地化
* :pencil2: - 修正错别字
* :hankey: - 编写需要改进的代码
* :alien: - 因外部API修改更新代码
* :truck: - 移动或重命名文件
* :page_facing_up: - 添加或更新许可证
* :boom: - 介绍重大变化
* :ok_hand: - 因代码审核更新代码
* :wheelchair: - 提升可访问性
* :bulb: - 为源码添加文档
* :beers: - 忘乎所以地编写代码
* :speech_balloon: - 更新文字
* :card_file_box: - 执行数据库相关的更改
* :loud_sound: - 添加日志
* :mute: - 删除日志
* :busts_in_silhouette: - 添加贡献者
* :children_crossing: - 改善用户体验或可用性
* :building_construction: - 进行架构更改
* :iphone: - 进行响应式设计
* :clown_face: - Mock相关
* :egg: - 添加一个复活节彩蛋
* :see_no_evil: - 添加或更新.gitignore
* :camera_flash: - 添加或更新快照
